# Assets - Imágenes

Esta carpeta contiene los logos y recursos gráficos de Ebonia.

## Logos requeridos para el login

Coloca los siguientes archivos en esta carpeta:

1. **`grupo-ebone-white.svg`** o `.png`
   - Logo de Grupo Ebone en **blanco**
   - Se muestra en el lado izquierdo (gradiente turquesa)
   - Tamaño recomendado: altura de ~64px

2. **`ebonia-logo.svg`** o `.png`
   - Logo de Ebonia en **colores** (turquesa + verde)
   - Se muestra en el formulario de login (lado derecho)
   - Tamaño recomendado: altura de ~64px

## Formato

Preferiblemente usa **SVG** para mejor calidad en todas las resoluciones.
Si usas PNG, asegúrate de que tenga buena resolución (al menos 2x).

## Rutas en el código

- `/assets/images/grupo-ebone-white.svg` → Panel izquierdo
- `/assets/images/ebonia-logo.svg` → Panel derecho (formulario)
